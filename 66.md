A company uses Amazon S3 to store files and images in a variety of storage classes. The company's S3 costs have increased substantially during the past year.  
  
A solutions architect needs to review data trends for the past 12 months and identity the appropriate storage class for the objects.  
  
Which solution will meet these requirements?

- A. Download AWS Cost and Usage Reports for the last 12 months of S3 usage. Review AWS Trusted Advisor recommendations for cost savings.
- B. Use S3 storage class analysis. Import data trends into an Amazon QuickSight dashboard to analyze storage trends.
- C. Use Amazon S3 Storage Lens. Upgrade the default dashboard to include advanced metrics for storage trends.
- D. Use Access Analyzer for S3. Download the Access Analyzer for S3 report for the last 12 months. Import the .csv file to an Amazon QuickSight dashboard.

## Analysis

### Understanding the Problem

The company is experiencing a **substantial increase in Amazon S3 costs** and needs to **review data trends for the past 12 months** to **identify the appropriate S3 storage class** for its objects. The core requirement is to analyze the storage usage patterns (trends) over an extended period to inform a strategy for cost optimization by correctly classifying storage. This falls under the domain of AWS cost management and S3 storage optimization.

### Correct Option(s) Explanation

**C. Use Amazon S3 Storage Lens. Upgrade the default dashboard to include advanced metrics for storage trends.**

- **Amazon S3 Storage Lens** is the **most appropriate and modern AWS tool** for gaining the required **organization-wide visibility** and deep insight into S3 usage and activity trends.
    
- It offers a **holistic view** across accounts and regions, which is often necessary in a large company environment.
    
- The requirement is to review **data trends** over 12 months to determine the **appropriate storage class**. S3 Storage Lens's **advanced metrics** (which is why the upgrade is mentioned) specifically include metrics like **retrieval requests** and **data usage trends** that are _essential_ for accurately determining access patterns and, thus, the optimal storage class (e.g., if objects are rarely accessed, they can move to S3 Glacier Flexible Retrieval).
    
- Using S3 Storage Lens directly addresses the need for **analytics on S3 storage trends**, which is a more direct and efficient solution than downloading and processing raw reports.
    

---

### Incorrect Options Explanation

**A. Download AWS Cost and Usage Reports for the last 12 months of S3 usage. Review AWS Trusted Advisor recommendations for cost savings.**

- **Incorrect Service Combination:** While the CUR contains the raw data, processing and analyzing 12 months of detailed S3 line-item data from the CUR to extract _storage class suitability trends_ is **highly complex and time-consuming**. It's not the intended or most efficient method.
    
- **Trusted Advisor** provides high-level cost-saving recommendations, but it does **not** provide the detailed, 12-month access pattern analysis required to make precise storage class decisions.
    

**B. Use S3 storage class analysis. Import data trends into an Amazon QuickSight dashboard to analyze storage trends.**

- **Suboptimal Tool:** **S3 Storage Class Analysis** _can_ do the job, but it operates at the **bucket or prefix level**. For a large company with substantial costs, **S3 Storage Lens** provides a far more scalable, centralized, and powerful organization-wide view and analysis. The option is technically possible but represents a less efficient and comprehensive solution than Storage Lens.
    

**D. Use Access Analyzer for S3. Download the Access Analyzer for S3 report for the last 12 months. Import the .csv file to an Amazon QuickSight dashboard.**

- **Wrong Service:** **Access Analyzer for S3** is a **security and access control** tool; its reports focus on public or cross-account access policies. It provides **no metrics or data** on storage usage, object access patterns, or cost trends, making it irrelevant for solving this problem.
